C# Gentle Formatter

VS Code extension that formats C# with proper indentation while preserving blank lines for clean git diffs.

v1.0.0
Download VSIX View on GitHub

Features

{ }

Proper Indentation

Automatically indents based on brace depth. Nested classes, methods, and blocks all get correct indentation.

_

Preserves Blank Lines

The key feature: blank lines keep their indentation. No more stripped whitespace causing noisy git diffs.

if

Keyword Spacing

Adds space after keywords: if(x) becomes if (x). Works with for, while, switch, catch, etc.

=

Operator Spacing

Normalizes operators: a=b+c becomes a = b + c. Handles ==, &&, ||, =>, ??, and more.

" "

String-Aware

Respects all C# string types: regular, verbatim @"", interpolated $"", and raw """ strings.

//

Comment-Safe

Never touches content inside comments. Both single-line // and multi-line /* */ are preserved exactly.

Why This Exists

Other C# formatters (OmniSharp, CSharpier) strip whitespace from blank lines. When teammates use different editors, this creates noisy git diffs that obscure the actual changes.

This formatter keeps your blank lines indented, so your diffs stay clean.

Other Formatters
public void Method()
{
    var x = 1;
                          // blank line stripped!
    var y = 2;
}
C# Gentle Formatter
public void Method()
{
    var x = 1;
    		              // indent preserved!
    var y = 2;
}

Usage

Shift + Alt + F Format document
Ctrl + Shift + P Command Palette: "C# Gentle Format"
editor.formatOnSave Auto-format when saving

Installation

1

Download the VSIX

Get the latest release from GitHub.

2

Install in VS Code

Ctrl+Shift+P → "Extensions: Install from VSIX..." → Select the file.

3

Set as Default Formatter

Add to your settings.json (see below).

Configuration

settings.json
{
    "[csharp]": {
        "editor.defaultFormatter": "trentsterling.csharp-gentle-formatter"
    },
    "csharpGentleFormatter.enabled": true,
    "csharpGentleFormatter.keywordSpacing": true,
    "csharpGentleFormatter.operatorSpacing": true
}
